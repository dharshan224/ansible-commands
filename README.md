# ansible-commands
## To Copy the FILE
- use the commands
  - present working directory
    ```
    ## pwd ansible dev -a "pwd"   
    ```
  - list
    ```
    pwd    ##present working directory
    ```
  - list -a
    ```sh
    ansible dev -a "pwd"                                                   
    /home/username
    ansible dev -a "ls"
    ansible dev -a "ls -a"
    ```
